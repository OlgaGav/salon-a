### Hair Salon A

Hair salon A is a responsive, static website built for a professional hair studio. 
The project focuses on clean design, performance, and real-world business needs such as multilingual support and booking system integration.

Live demo: https://olgagav.github.io/salon-a/

Repository: https://github.com/OlgaGav/salon-a/


## Features

- Modern, elegant UI tailored for a beauty salon brand
- Fully responsive layout (desktop, tablet, mobile)
- Multi-language support (English / Russian)
- Integrated booking flow (ready for third-party services such as Square Appointments)
- Service descriptions and stylist profile sections
- Instagram integration for social visibility
- Semantic HTML optimized for SEO and accessibility
- Lightweight, fast-loading implementation (no frameworks)
